Bootstrap

Om Bootstrap

Bootstrap är ett mycket populärt ramverk som går att använda med enbart CSS eller med CSS + JavaScript (jQuery för version 4 och neråt). Det är nybörjar-vänligt med en omfattande dokumentation och ett utbrett community att fråga om hjälp.

Bootstrap innehåller bland annat grid-system och färdiga komponenter så som knappar. Det är responsivt och håller måttet i olika webbläsare. Det finns också extra verktyg för att lägga till animeringar et cetera.

Det är möjligt att göra egna modifieringar i utseendet med hjälp av SASS/CSS.

Mina åsikter

Jag tycker det var väldigt smidigt att jobba med, när man väl kom in i tänket. Speciellt användbart var den inbyggda funktionaliteten för modals, alerts osv. Det var enkelt att använda då man inte behövde skriva någon egen JavaScript-kod, perfekt för en nybörjare!

Nackdelen är att ramverket känns lite stelt. Det låter en inte göra några större utsvävningar, speciellt om man inte vill ge sig in och skriva SASS.

Jag skulle kunna tänka mig att arbeta mer med Bootstrap, men kanske framför allt till mindre uppgifter och admingränssnitt osv. Jag tycker själva designbiten är på tok för rolig för att låta ett ramverk sköta så stora delar av den i projekt som faktiskt spelar roll.
